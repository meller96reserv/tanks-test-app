# Tanks

React + TypeScript app that displays a paginated, filterable table of tanks from the Tanks API.

## Requirements

- Node.js ≥ 16
- npm ≥ 8

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000).

## Tests

```bash
npm test
```

To run once without watch mode:

```bash
npm test -- --watchAll=false
```

## Build

```bash
npm run build
```

The production build will be output to the `build/` folder.
